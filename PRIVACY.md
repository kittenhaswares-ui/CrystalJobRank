# Short privacy notice

Last updated: 15 July 2026

Crystal Job Rank keeps the full match history, all ten scoreboard rows, local
records, and achievements on your PC.

For the automatic community leaderboard, each future Casual or Ranked result
sends only:

- your character name, numeric Home World ID, and the Home World label read from
  the game (the service derives the public World name from the numeric ID);
- your job, win or loss, queue type, completion time, and a deterministic
  duplicate-detection key made only from your own result;
- your own result-screen values for basic validation; and
- a random installation key generated by the plugin.

The installation key is not a Square Enix account ID, character content ID,
hardware ID, or login. Neither it nor its SHA-256 digest is stored in the
leaderboard database; the digest is used transiently for request rate limits.
It never locks or links a character to an installation. Other players' names,
worlds, IDs, or scoreboard values are never uploaded. Raw personal combat
values, territory, and duration are validated in transit but are not retained
by the hosted leaderboard.

Public leaderboard rows contain `Character Name · Home World`, job, rating,
matches, wins, losses, and win rate. The board is cross-region; region and data
center are not collected.

The reference API uses HTTPS and runs on Cloudflare Workers with an
EU-jurisdiction D1 database. The application does not intentionally log request
bodies, installation keys, character names, or IP addresses. Cloudflare may
process normal connection and security metadata as the hosting provider.

This is an unofficial, community-reported board. A modified client can submit
false results because no official match-attestation API exists. For a data
removal request or privacy problem, open a repository issue without posting
secrets: https://github.com/kittenhaswares-ui/CrystalJobRank/issues
